# Python & ML Revision Repository

Welcome to the revision workspace! This repository is set up to help you practice and test your Python skills.

## How to Proceed

Please follow these steps in order to get the most out of your revision:

1. **Read the revision guide**: Start by going through `revision_guide.py` to refresh your knowledge on the core concepts.
2. **Finish Level 1**: Go to the `level1` directory and finish the projects there to build a strong foundation.
3. **Finish Level 2**: Once you're done with the first level, go to the `level2` directory and tackle the more advanced projects.

## Practice Guidelines

* 📝 **Cheatsheet**: You can use the `cheatsheet.md` while coding if you need a quick syntax or concept reference.
* 🚫 **No AI Assistance**: Please **turn off all AI coding tools** while you are coding so you can practice well and rely on your own problem-solving skills.

Happy coding and good luck with your revision!
